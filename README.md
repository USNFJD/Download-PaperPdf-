# OA PDF 批量下载器

这是一个本地 WebView 桌面程序。Python/FastAPI 在后台运行，界面嵌入到 WebView 窗口里。

## 运行

推荐双击：

```text
start_webview.bat
```

也可以在 PowerShell 里调试运行：

```powershell
powershell -ExecutionPolicy Bypass -File .\run.ps1
```

## 使用流程

1. 在 5 个独立输入框里填写关键词，空框会自动忽略。
2. 在“显示篇数”下拉框选择 100、200、300、400 或 500，默认 100。
3. 点击“检索”，程序会先展示候选 OA 文献，不会立即下载。
4. 检索记录默认按关联度优先、发表时间次之排序；点击“文献名称”恢复这个排序，点击“发表时间”则按发表时间优先、关联度次之排序。
5. “关联度”列会显示 0% 到 100% 的相对关联度，颜色从淡绿到深绿。
6. 在候选列表最后一列勾选文献，选择仓库 1、仓库 2 或仓库 3，然后点击“添加勾选”。
7. 点击“加载仓库”可以查看仓库记录；仓库内序号按添加先后固定显示，删除某一条后会重新排序。
8. 点击“下载仓库”只会下载当前仓库里的文献，不会下载候选列表里的全部文献。

## 关键词组合规则

程序先执行两两组合检索：

- 5 个关键词会形成 10 个两两组合。
- 如果关键词少于 5 个，则按实际非空关键词生成组合。
- 如果两两组合凑不够目标篇数，会继续用单个关键词补充检索。
- 5 个关键词时最多 15 个检索组合：10 个两两组合 + 5 个单关键词。

## 文件位置

- PDF：`downloads\`
- 数据库：`data\papers.sqlite3`
- 日志：`data\app.log`
- SCI 分区表：`data\journal_quartiles.csv`

## SCI 分区

JCR/SCI 分区数据不是开放数据库，本项目不会内置或抓取受限数据。请把你有授权的分区表保存到：

```text
data\journal_quartiles.csv
```

支持列名：`issn_l`、`issn`、`journal_title`、`journal`、`sci_quartile`、`quartile`、`jcr_quartile`、`中科院分区`。界面点击“重载分区”即可重新匹配。
